#### Required software
* NodeJS
* NPM


#### Install the packages required to build
Inside the cloned directory, run
```
npm install
```

#### Compile the TypeScript to JavaScript
```
npm run-script build
```

#### Interactively run the "CPU"

```
npm run-script run
```
